## ISAI – Full Stack Music Player (MERN Stack)
A full-featured Music Player Web Application built using the MERN Stack. This project demonstrates complete frontend–backend integration with authentication, API handling, state management, and responsive UI design.

## Project Overview

ISAI is a modern music streaming web application that allows users to:
-  Create an account and securely log in
- Search for songs
-  Play, pause, and control music
- View song details
- Manage user profile
- Maintain personalized playlists
- Add/remove favourite songs
 
## Features
- Authentication System

  User Signup & Login

  JWT-based authentication

  Protected routes using middleware

  Reset Password via Email (Nodemailer)

  Profile Editing functionality

- Music Player Features

  Play / Pause audio

  Next / Previous track

  Song progress control

  Song details display

  Dynamic playlist rendering

  Reusable custom audio hook

- Search Functionality

  Search songs by tags

  API-based song fetching

  Real-time result rendering

  Error handling for API responses

- User Interaction Features

  Add to Favourites

  Remove from Favourites

  Playlist creation & viewing

  Persistent state using Redux

## Technologies Used

- Frontend

  React.js (Vite)

  JavaScript (ES6+)

  Redux Toolkit

  Axios

  Tailwind CSS

  Custom React Hooks

- Backend

  Node.js

  Express.js

  MongoDB

  Mongoose

  JWT (JSON Web Token)

  Nodemailer
